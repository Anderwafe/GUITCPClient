## Installation

### From Releases
Go to `Releases` page and download `.tar.xz` archive.
Unpack it with anything what you love.

### From Sources
#### Prerequisites
1. .NET 8.0 SDK

#### Download sources from github
```shell
git clone https://github.com/Anderwafe/GUITCPClient.git
```

#### Build and launch
```shell
cd GUITCPClient
dotnet build
dotnet run
```
